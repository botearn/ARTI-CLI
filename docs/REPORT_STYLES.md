# ARTI CLI 报告样式对比

本文档展示 **全景报告**（`arti full`）和 **深度研报**（`arti deep`）两种模式的输出样式差异。

## 快速对比

| 维度 | 全景报告（arti full） | 深度研报（arti deep） |
|------|---------------------|----------------------|
| **图标** | 📊 | 📈 |
| **Layer 1** | 5 位分析师（Natasha → Steve/Tony/Clint → Thor） | 8 位分析师全套（+ Sam/Vision/Wanda） |
| **Layer 2** | 2-4 位大师（sector 路由） | 7 位大师完整辩论 |
| **Synthesizer** | panorama_synthesizer（压缩总结） | synthesizer（深度裁定） |
| **耗时** | 3-5 分钟 | 20+ 分钟 |
| **计费** | 30 Credits | 100 Credits |
| **使用场景** | 快速全景扫描 · 日常监控 | 重仓决策 · 投委会报告 |

## 关键差异总结

### 1. **图标与标题**

- 全景：📊 全景报告
- 深度：📈 深度研报

### 2. **分析师数量**

- 全景：5 位（Natasha → Steve/Tony/Clint → Thor）
- 深度：8 位（+ Sam/Vision/Wanda）

### 3. **大师辩论**

- 全景：2-4 位（sector 路由），显示 💬 图标，标注"轻量辩论"
- 深度：7 位完整辩论，显示 🎯 图标，标注"完整辩论"

### 4. **Synthesis 输出**

#### 全景（压缩总结）
- 多头联盟 / 空头质疑
- 核心分歧
- 综合评级（短期/中期/长期）
- 失败信号（3 条）

#### 深度（深度裁定）
- 七位共识点
- 核心分歧点
- 风险收益比评估（盈亏比计算）
- 仓位建议（分流派）
- 关键变量灵敏度（条件 → 影响）
- 观察时间节点（财报日 / 催化剂）
- 失败信号（5+ 条，更详细）

### 5. **用途场景**

- 全景：快速全景扫描 · 日常监控 · 趋势判断
- 深度：重仓决策 · 深度研究 · 投委会报告

### 6. **计费与耗时**

- 全景：30 Credits，3-5 分钟
- 深度：100 Credits，20+ 分钟

## 使用建议

### 何时用全景报告？

1. 快速了解一只股票的整体情况
2. 日常投资组合监控
3. 寻找新的投资机会
4. 需要快速决策时

### 何时用深度研报？

1. 准备重仓某只股票
2. 需要向投委会汇报
3. 深度研究特定标的
4. 需要量化评估风险收益比
5. 需要明确的仓位建议

### 配合使用策略

```bash
# 第一步：快速扫描（全景）
arti full AAPL

# 如果全景结果看好 → 第二步：深度研究
arti deep AAPL --full

# 需要完整报告详情时加 --full 选项
arti full AAPL --full
arti deep AAPL --full
```

## 技术实现

两种模式通过以下参数区分：

```typescript
// 全景报告
researchCommand(symbol, { mode: "panorama", full: false });

// 深度研报
researchCommand(symbol, { mode: "deep", full: false });

// 后端 mode 映射
"panorama" → "layer1-only" → Layer 1 only
"deep"     → "full"        → Layer 1 + Layer 2 + Synthesis
```

**注意**：实际后端只有两种模式 `full` 和 `layer1-only`，但前端通过分析师数量和大师数量判断是全景还是深度，从而显示不同的样式和提示。
